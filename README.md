# FleetOps

FleetOps is a fleet management platform designed to help companies manage vehicles, maintenance records, fuel consumption and operational metrics.

## Features

- User authentication
- Vehicle management
- Fuel records
- Maintenance records
- Fleet dashboard
- Operational KPIs

## Tech Stack

### Backend

- Java 21
- Spring Boot 3
- Spring Security
- PostgreSQL
- Flyway

### Frontend

- React
- TypeScript
- Vite
- Material UI

### DevOps

- Docker
- GitHub Actions
