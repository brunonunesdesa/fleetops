# FleetOps - UI/UX Documentation

## Design Philosophy

FleetOps follows a modern SaaS dashboard approach inspired by:

* Stripe Dashboard
* Linear
* Notion

Goals:

* Simple navigation
* Fast access to operational information
* Enterprise-grade appearance
* Responsive design

---

# Screens

## Login

### Purpose

Allow users to authenticate into the platform.

### Components

* FleetOps logo
* Email input
* Password input
* Sign In button
* Forgot Password link

---

## Dashboard

### Purpose

Provide a high-level overview of fleet operations.

### KPI Cards

* Total Vehicles
* Active Vehicles
* Vehicles In Maintenance
* Fuel Costs

### Charts

* Monthly Expenses
* Fleet Status

### Tables

* Recent Maintenance

---

## Vehicles

### Purpose

Manage registered vehicles.

### Features

* Search vehicles
* Filter by status
* Create vehicle
* Edit vehicle
* Delete vehicle

### Columns

* Vehicle ID
* Model
* License Plate
* Mileage
* Last Service
* Status
